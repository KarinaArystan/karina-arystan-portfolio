## Technical Notes

### Commands
- `npm run dev` ‚Äî Start dev server (port 4321)
- `npm run build` ‚Äî Production build to `dist/`
- `npm run preview` ‚Äî Preview production build

### Dependencies
- **Astro 6.1.5** ‚Äî Static site generator
- **Tailwind CSS 4** ‚Äî Utility-first CSS (via `@tailwindcss/vite` plugin)
- **GSAP 3.12** ‚Äî ScrollTrigger for reveal animations
- **Lenis 1.1** ‚Äî Smooth scroll (integrated with GSAP ticker)

### Fonts
- **Fraunces** (variable, serif) ‚Äî Headings (`font-heading`)
- **Plus Jakarta Sans** (variable, sans) ‚Äî Body text (`font-body`)
- Loaded via Google Fonts in `BaseLayout.astro`
